O que é HTML ?

É uma linguagem de marcação de texto (Hypertext Markup Language)

Comentários no HTML (< !--algo que queira deixar como comentario.-->)

Comentários no HTML é usado com esses simbolos que estão entre parenteses( < !-- --> )

Atributos

Pegar a tag img, que contém atributos nela, para servir de exemplo. ( img (src="" alt="") )

Atributos Globais

- Class / - contendtable / - data-* / - hidden / - id / - style / - tabindex / - title /

exemplo da tag Class: < div class="nomedaclasse">
a tag class é usado para colocar estilo no CSS e acessar também no JS(JavaScript), mais de um elemento pode ter o nome da mesma classe.

exemplo da tag Contendtable: < div class="nomedaclasse" contenteditable="true" > Conteudo< /div>
ele significa que eu posso editar o conteúdo que ele está recebendo na propria página, mas... assim que atualizar a página a edição some.

exemplo da tag data-qualquer-nome-aqui: < div class="nomedaclasse" data-id="">
e muito usado no JS, ele consegue expandir que tipo de atributo eu posso colocar no meu html.

exemplo da tag hidden: < div class="nomedaclasse" hidden >
ele é usado para esconder uma tag

exemplo da tag Id: < div class="nomedaclasse" id="" >
pode-se usar apenas 1 por página, e não pode colocar um id com o mesmo nome, como se faz com a tag class.

exemplo da tag Style: < div class="nomedaclasse" style="color:red">
usa-se para aplicar elementos do css dentro de uma propria tag, geralmente não se é usado dessa forma.

exemplo da tag tabindex: < div class="nomedaclasse" tabindex="1" >
ela é usada para navegar pela página com a tecla Tab.

exemplo da tag title: < div class="nomedaclasse" title="">
Apenas para definir um titulo.
